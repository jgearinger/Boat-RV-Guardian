Boat & RV Guardian

A free, open-source water safety dashboard for LinkTap. Monitor flow rate and automatically shut off your water supply to prevent flooding.

Open Web Demo Download App

Project Overview

Boats and RVs are susceptible to water damage from burst pipes when connected to municipal water supplies. Boat & RV Guardian serves as an automated safety sentry using the LinkTap API.

Instead of relying purely on hardware limits, the Guardian app runs on your device and sends a continuous "heartbeat" to the valve. If a pipe bursts and water usage exceeds your configured safe limits (duration or volume), the valve shuts off. If your device loses internet or the app is closed, the LinkTap hardware's fail-safe kicks in and shuts the valve down automatically.

Hardware Requirements

To use all features of the Guardian app, specific LinkTap hardware is required:

  • LinkTap Gateway (GW-01 or GW-02): Required to bridge your home Wi-Fi network to the LinkTap Valve and support the necessary Local API endpoints.
  • LinkTap G2S Valve (TapLinker): Must be the G2S (or newer G1S/G4S) model because it includes a built-in flow meter. Older generations (like the G1) lack flow meters and cannot be used for volume-based auto-shutoffs.
  • Water Pressure Regulator: (Recommended) Always install a standard RV/Boat water pressure regulator before the LinkTap valve to protect the smart valve from high-pressure dock water.

Setup Instructions

Step 1: LinkTap Hardware Setup

  1. Follow the official LinkTap instructions to connect your Gateway to your home Wi-Fi network.
  2. Insert batteries into the LinkTap G2S Valve and pair it with the Gateway using the official LinkTap mobile app.
  3. Verify that you can manually turn the water on and off using the official LinkTap app before proceeding.

Step 2: Connect via Local API (Recommended)

Because LinkTap does not expose Cloud API keys in their app or portal (they must be explicitly requested from LinkTap Support), the Local HTTP API is the primary and recommended way to use Guardian. It provides internet-free reliability and immediate response times.

Note: Due to browser security (CORS), the Local API is only supported using our native Desktop or Android apps.

  1. Open the official LinkTap mobile app.
  2. Go to Gateway Details and enable the Local HTTP API toggle.
  3. Download the Boat & RV Guardian native app for your platform (see Downloads below).
  4. In the Guardian App settings, ensure API Mode is set to Local Network and enter your Gateway's local IP address (found in the LinkTap app).
  5. Enter your Gateway ID and Valve ID (Taplinker ID) (these are 16-character codes). Save the settings.

Step 3: Connect via Cloud API (Web Dashboard)

If you prefer to use the web-based dashboard and have successfully requested a Cloud API Key from LinkTap Support, you can connect via the cloud.

  1. Open the Web Dashboard.
  2. Click the Settings (⚙️) icon in the top right.
  3. Switch the API mode to Cloud API.
  4. Enter your LinkTap Cloud Username and your provided API Key.
  5. Enter your Gateway ID and Valve ID (Taplinker ID) and save the settings.
