Project Overview

Boats and RVs are susceptible to water damage from burst pipes when connected to municipal water supplies. Boat & RV Guardian serves as an automated safety sentry using the LinkTap API.

A burst pipe while connected to city water can cause catastrophic flooding if left unchecked.

Instead of relying purely on hardware limits, the Guardian app runs on your device and sends a continuous "heartbeat" to the valve. If a pipe bursts and water usage exceeds your configured safe limits (duration or volume), the valve shuts off. If your device loses internet or the app is closed, the LinkTap hardware's fail-safe kicks in and shuts the valve down automatically.

Why LinkTap vs. Tuya-based Valves?

There are many inexpensive Wi-Fi water valves on the market (often white-labeled through the Tuya smart home platform). However, they are fundamentally inadequate for critical safety applications like Boat/RV flood prevention for several reasons:

❌ Standard Smart Valves (Tuya)

  • No Flow Meter: They can only measure time (e.g., "open for 10 minutes"). If a pipe bursts, they have no idea that hundreds of gallons are flowing out.
  • Wi-Fi Battery Drain: Connecting directly to Wi-Fi burns battery rapidly, meaning they require frequent battery swaps or AC power.
  • Cloud Dependency: Commands must travel to a remote cloud server and back. If your campground Wi-Fi drops, the valve cannot be controlled.
  • No Hardware Fail-safes: If the valve loses connection while open, it stays open indefinitely until commanded otherwise.

✅ LinkTap G2S System

  • Built-in Flow Meter: The G2S actively measures exact water volume and flow rate. This allows the Guardian app to shut off water based on actual gallons consumed.
  • Zigbee Protocol: The valve communicates over low-power Zigbee to the indoor Gateway, ensuring extreme battery life (up to 2 years) and better penetration through metal/fiberglass RV walls.
  • Local API Support: Guardian talks directly to the Gateway on your local network. No internet access is required, ensuring instant sub-second response times.
  • Hardware Timers: Every time the valve opens, a strict duration is sent to the hardware. If the Guardian app or network crashes, the valve's physical hardware timer will forcefully close the valve.

Setup Instructions

Step 1: LinkTap Hardware Setup

  1. Follow the official LinkTap instructions to connect your Gateway to your home Wi-Fi network.
  2. Insert batteries into the LinkTap G2S Valve and pair it with the Gateway using the official LinkTap mobile app.
  3. Verify that you can manually turn the water on and off using the official LinkTap app before proceeding.

Step 2: Connect via Local API (Recommended)

Because setting up a Cloud API Key requires logging into the LinkTap web portal on a computer, the Local HTTP API is the primary and recommended way to use Guardian. It provides internet-free reliability and immediate response times.

Note: Due to browser security (CORS), the Local API is only supported using our native Desktop or Android apps.

  1. Open the official LinkTap mobile app.
  2. Go to Gateway Details and enable the Local HTTP API toggle.
  3. Download the Boat & RV Guardian native app for your platform (see Downloads below).
  4. In the Guardian App settings, ensure API Mode is set to Local Network and enter your Gateway's local IP address (found in the LinkTap app).
  5. Enter your Gateway ID and Valve ID (Taplinker ID) (these are 16-character codes). Save the settings.

Step 3: Connect via Cloud API (Web Dashboard)

If you prefer to use the web-based dashboard, you can generate a Cloud API Key by logging into the LinkTap API for Developers page. Then connect via the cloud:

  1. Open the Web Dashboard.
  2. Click the Settings (⚙️) icon in the top right.
  3. Switch the API mode to Cloud API.
  4. Enter your LinkTap Cloud Username and your provided API Key.
  5. Enter your Gateway ID and Valve ID (Taplinker ID) and save the settings.
