How and Why It Works

Boats and RVs are extremely susceptible to water damage from burst pipes when connected to "city water" or dock supply. Traditional water timers aren't designed for this. Boat & RV Guardian acts as a smart proxy for the LinkTap smart valve.

By connecting directly over your Local Wi-Fi Network to the LinkTap Gateway, the Guardian app sends automated, continuous "heartbeat" signals. If a pipe bursts and water runs longer than your configured safe limit, the valve automatically snaps shut. If your internet goes down, or if the dashboard is closed, the system falls back to safety and shuts off the water.

System Components

1. LinkTap Valve (TapLinker)

A heavy-duty, motorized brass water valve installed between your hose and your RV/Boat. It shuts off the water supply mechanically.

2. LinkTap Gateway

The central hub that talks to the valve via Zigbee and connects to your local Wi-Fi router. It handles the API requests from the Guardian app.

3. Flow Meter (Optional)

Measures exact gallons/liters used. The Guardian app can use this data to shut off the water after a precise volume limit is reached.

4. Boat & RV Guardian App

This software. A persistent dashboard that overrides the LinkTap's default behavior, keeping the water flowing as long as the app is running and no limits are breached.

Full Setup Directions (Start to Finish)

Step 1: Unbox & Setup LinkTap

  1. Follow the official LinkTap instructions to plug in the Gateway to power and connect it to your Wi-Fi network.
  2. Insert batteries into the TapLinker (Valve) and pair it with the Gateway using the official LinkTap mobile app.
  3. Ensure you can turn the water on and off using the official LinkTap app.

Step 2: Enable Local API (Crucial for Reliability)

For the fastest response time and internet-free reliability, enable the Local HTTP API:

  1. Open the LinkTap official app. Go to Gateway Details.
  2. Find the Local Network API setting and turn it ON.
  3. Note down the Gateway IP Address (e.g., 192.168.1.150).
  4. Note down the 16-character Gateway ID.
  5. Note down the 16-character TapLinker ID (Device ID).

Step 3: Configure the Guardian App

  1. Open the Guardian Web Dashboard (or install the native Android/macOS app).
  2. Click the Settings ⚙️ icon in the top right.
  3. Disable Mock Mode (Simulate Locally).
  4. Enter the Gateway IP Address, Gateway ID, and Device ID you collected earlier.
  5. Close the settings. The dashboard should immediately connect and display your valve's live status!

Dashboard Options Explained

Normal Run Mode

This is the standard mode for continuous protection. Set a Target Time (e.g., 20 minutes) and a Volume Limit (e.g., 30 Gallons). The Guardian app will tell the valve to stay open. Every few seconds, the app resets the timer. If the app crashes, or Wi-Fi drops, the valve will count down from 20 minutes and automatically shut off. Auto-Restart (Loop) automatically restarts the 20-minute countdown when the time runs out, meaning water stays on indefinitely as long as the app is running.

Fill a Tank / Custom Run

Need to fill your RV's fresh water tank with exactly 40 gallons? Enter 40 in the volume box and hit Start Custom Run. The valve will turn on and automatically shut off the moment 40 gallons have passed through the flow meter.

Mock Mode (Simulation)

Don't have the hardware yet? Turn on Mock Mode in settings. The app will simulate a virtual gateway so you can test how the timers, auto-restarts, and usage history features work without real water flowing.
