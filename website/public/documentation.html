System Layout & Architecture

Physical Plumbing Layout

This diagram illustrates the standard plumbing sequence. Crucially, the LinkTap G2S valve must be installed right next to the city water spigot (the source), rather than at the RV/Boat inlet. This ensures that the entire length of your drinking water hose is protected and will be shut off if it bursts.

Setup Instructions

Step 1: LinkTap Hardware Setup

Note: The LinkTap API does not support initial hardware pairing. You must use the official LinkTap app for this first step.

  1. Follow the official LinkTap instructions to connect your Gateway to your home Wi-Fi network.
  2. Insert batteries into the LinkTap G2S Valve and pair it with your Gateway using the official LinkTap mobile app.
  3. Verify that you can manually turn the water on and off using the official LinkTap app before proceeding.

Step 2: Connect via Local API (Recommended)

Because setting up a Cloud API Key requires logging into the LinkTap web portal on a computer, the Local HTTP API is the primary and recommended way to use Guardian. It provides internet-free reliability and immediate response times.

Note: Due to browser security (CORS), the Local API is only supported using our native Desktop or Android apps.

  1. Open the official LinkTap mobile app.
  2. Go to Gateway Details and enable the Local HTTP API toggle.
  3. Download the Boat & RV Guardian native app for your platform from the homepage.
  4. In the Guardian App settings, ensure API Mode is set to Local Network and enter your Gateway's local IP address (found in the LinkTap app).
  5. Enter your Gateway ID and Valve ID (Taplinker ID) (these are 16-character codes). Save the settings.

Step 3: Connect via Cloud API (Optional)

If you prefer to connect via the cloud instead of the local network, you can generate a Cloud API Key by logging into the LinkTap API for Developers page. Then connect via the cloud:

  1. Open the App.
  2. Go to Settings > Hardware Connections.
  3. Switch the API mode to Cloud API.
  4. Enter your LinkTap Cloud Username and your provided API Key, then click 📡 Auto-Discover Devices.
  5. Select your Gateway and TapLinker from the new dropdowns, and click ▶ Apply & Connect.
